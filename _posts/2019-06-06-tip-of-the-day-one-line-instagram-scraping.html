Instagram public data scraping

If for some reason you want to get public Instagram data for specific account - it's easy to do!

{% highlight ruby %} curl https://www.instagram.com/instagram/ \ | grep '_sharedData' \ | awk '{print substr($0,64,length($0)-11)}' \ | jq '.entry_data.ProfilePage[0].graphql.user' {% endhighlight %}

Notes:

  1. No login is required
  2. Substitute any {username}, you are interested in https://www.instagram.com/{username}/
  3. Further data mapping/filtering could be done with jq.
